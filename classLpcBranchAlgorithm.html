Detailed Description

Class that follows the scaled data points to find lpc branches.

This is a C++ implementation of the branch finding algorithm in the R-code package LPCM written by Jochen Einbeck and Ludger Evers (http://cran.r-project.org/web/packages/LPCM/index.html)
