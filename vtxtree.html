Vertex information stored in the vtxTree TTree

The table below describes the TBranch structure of the TTree labelled vtxTree generated by LpcRootOutput, which stores information contained in LpcVertex objects.

Each entry in vtxTree corresponds to one LpcVertex.

Quantity Type

Description

eventId integer

LpcEvent index number

nVtx integer

The number of vertices in the LpcEvent

vtxIndex integer

The index number of the LpcVertex

vtxCurveId integer

The index number of the main LpcCurve associated to this LpcVertex

vtxBranchId integer

The index number of the LpcBranch associated to this LpcVertex (main curve = 0)

vtxXi double

The ith co-ordinate component of the LpcVertex position
