LpcRootInput TBranch structure

The LpcRootInput class can be used to read in a ROOT file of the dataset, containing a TTree with the name "Data", with the following TBranch structure:

Variable name Variable type

Variable description

eventId integer

The LpcEvent index number

nHits integer

The number of hits for the given LpcEvent

hitX1 vector <double>

The vector of the first co-ordinate component for all LpcHits

hitX2 vector <double>

The vector of the second co-ordinate component for all LpcHits

hitX3 vector <double>

The vector of the third co-ordinate component for all LpcHits

... ...

...

hitXn vector <double>

The vector of the Nth co-ordinate component for all LpcHits

hitW vector <double>

The vector of the weights for all LpcHits
