Cluster information stored in the clsTree TTree

The table below describes the TBranch structure of the TTree labelled clsTree generated by LpcRootOutput, which stores information contained in LpcCluster objects. The size of the STL vector objects are given by the preceeding integer variable appearing in the branch.

Each entry in clsTree corresponds to one LpcCluster.

Quantity Type

Description

eventId integer

LpcEvent index number

nCl integer

The number of LpcClusters in the LpcEvent

clIndex integer

Index number of the LpcCluster

clCurveId integer

Index number of the main curve associated to the LpcCluster

clBranchId integer

Index number of the branch associated to the LpcCluster (0 is the main curve)

clMinLpc integer

The lowest index number of the LpcPoints used for the LpcCluster

clMaxLpc integer

The highest index number of the LpcPoints used for the LpcCluster

clXi double

The ith co-ordinate component of the LpcCluster centroid position

nAxes integer

The number of co-ordinate (principal) axes of the LpcCluster

clAxisXi vector<double>
size=nAxes

The ith co-ordinate components of the principal axes of the LpcCluster

clHullXi vector<double>
size=nAxes

The lengths along the ith co-ordinate axis of the convex hull encompassing the boundary of hits in the LpcCluster, after the hit positions have been rotated with respect to the main principal axis of the cluster

clShower integer

Specify if the LpcCluster is identified as a shower (1) or not (0)

nClHits integer

The number of LpcHits within the LpcCluster

clHitXi vector<double>
size=nClHits

The ith co-ordinate component of the LpcHits within the LpcCluster

clHitW vector<double>
size=nClHits

The weights of the LpcHits within the LpcCluster
