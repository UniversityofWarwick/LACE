Detailed Description

Class that follows the scaled data points to find the lpc.

This is a C++ implementation of the main Lpc curve algorithm in the R-code package LPCM written by Jochen Einbeck and Ludger Evers (http://cran.r-project.org/web/packages/LPCM/index.html)
