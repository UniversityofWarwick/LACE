Curve information stored in the lpcTree TTree

The table below describes the TBranch structure of the TTree labelled lpcTree generated by LpcRootOutput, which stores information contained in LpcAbsCurve objects. The size of the STL vector objects are given by the preceeding integer variable appearing in the branch.

Each entry in lpcTree corresponds to one curve (either LpcCurve or LpcBranch).

Quantity Type

Description

eventId integer

LpcEvent index number

curveId integer

Index number of the curve

branchId integer

Index number of the branch (main curve = 0)

branchGen integer

The branch generation number (main curve = 0)

nLpc integer

Number of LpcPoints

lpcInitXi double

The ith co-ordinate component of the starting point of the curve

lpcXi vector<double>
size=nLpc

The ith unscaled co-ordinate component of each LpcPoint

eigenXi vector<double>
size=nLpc

The ith co-ordinate component of the largest eigenvector for each LpcPoint

cosAngles vector<double>
size=nLpc

Cosine of the angle between the largest eigenvector of each LpcPoint with the eigenvector of the proceeding LpcPoint

rho vector<double>
size=nLpc

Ratio of the 2nd largest to the largest eigenvalues λ2/λ1 for each Lpc point

c0 vector<double>
size=nLpc

Adjusted kernel function denominator exponent scale factor for each Lpc point

lambda vector<double>
size=nLpc

Cumulative signed unscaled pathlength position of each LpcPoint from the Lpc starting point

lambdaXi vector<double>
size=nLpc

The ith co-ordinate component of the cumulative signed unscaled pathlength position of each LpcPoint from the Lpc starting point

pathLength vector<double>
size=nLpc

Absolute unscaled cumulative pathlength position of each LpcPoint

lpcResiduals vector<double>
size=nLpc

Average value of all of the residuals from the closest LpcHits for each LpcPoint

wLpcResiduals vector<double>
size=nLpc

Average value of all of the residuals from the closest LpcHits for each LpcPoint, weighted by the hit weights

nHighRho integer

The number of LpcPoints that have rho values above the value rho0 for possible branching points

highRhoXi vector<double>
size=nHighRho

The ith co-ordinate component of the LpcPoint with a "high" rho value, i.e. above rho0

nCosPeaks integer

The number of LpcPoints that correspond to peaks in the 1-|cosAngle| distribution

cosPeakId vector<int>
size=nCosPeaks

The index number of the LpcPoints corresponding to the peaks in the 1-|cosAngle| distribution (in descending height order)

nHits integer

The number of hits in the LpcEvent

hitXi vector<double>
size=nHits

The ith co-ordinate component of each LpcHit

hitResiduals vector<double>
size=nHits

The residual distance of the closest LpcPoint for each LpcHit

wHitResiduals vector<double>
size=nHits

The residual distance of the closest LpcPoint for each LpcHit multiplied by the hit weight

hitNearestLpc vector<int>
size=nHits

The index number of the closest LpcPoint for each LpcHit
