Text output format

The output given by LpcTextOutput is shown in the table below.

Quantity

Description

Event

The event index number integer

MainCurve:Id

The index number integer of the main LpcCurve

LpcStartPoint

The unscaled co-ordinates of the starting point of the LpcCurve

NumberOfLpcPoints

The integer number of points in the LpcCurve

LpcPoint

Each line gives the integer index number, then the unscaled co-ordinates of each LpcPoint in the LpcCurve

LpcEigenVectors nLpc nDim

The nDim components of the main, largest eigenvector of each Lpc point (1 per line). Here, nLpc is the total number of points and nDim the number of co-ordinate dimensions

LpcCosineAngles nLpc

Cosine of the angle between the main eigenvector of the given Lpc point with the main eigenvector of the previous point (10 points per line)

EigenRatioRho nLpc

Ratio of the 2nd largest to the largest eigenvalues λ2/λ1 for each Lpc point (10 points per line)

c0 nLpc

Value of the adjusted kernel function denominator exponent scale factor for each Lpc point (10 points per line)

HighRhoPoints

The number of Lpc points, followed by the Lpc points themselves, that have rho values above the value rho0 for possible branching points. This may be missing if no points satisfy rho > rho0

LpcLambda nLpc

Cumulative signed unscaled pathlength position of each LpcPoint from the Lpc starting point (10 points per line)

LpcLambdaAxes nLpc nDim

The nDim components of the signed cumulative unscaled pathlength from the Lpc starting point for each Lpc point (1 point per line)

LpcPathLength nLpc

Absolute unscaled cumulative pathlength position of each LpcPoint along the curve (10 points per line)

MeanLpcResiduals nLpc

For each LpcPoint (10 per line), the average value of the residual distance from the closest LpcHits

HitResiduals nHits

For each LpcHit (10 per line, nHits in total), the value of the residual distance of the closest LpcPoint

WeightMeanLpcResiduals nLpc

For each LpcPoint (10 per line), the average value of the residual distance of the closest LpcHits, weighted by the hit weights

WeightHitResiduals nHits

For each LpcHit (10 per line, nHits in total), the value of the residual distance of the closest LpcPoint multiplied by the hit weight

HitNearestLpc nHits

For each LpcHit (10 per line, nHits in total), the integer index number of the nearest LpcPoint

CosPeaks nPeaks

Lists the indices of the LpcPoints that correspond to the peaks in the 1 - |cosAngle| distribution (in descending height order)

BranchId

The index number integer of the LpcBranch (= 0 for the main curve)

Branch:MainCurveId

The index number integer of the main LpcCurve associated to this LpcBranch

Branch:GenerationLevel

The integer specifying the generation level of the LpcBranch (starting from 1). The same Lpc information detailed above is then displayed for this branch

NumberOfVertices

The integer number of vertices found for the dataset

Vertex:Index

The integer index number of the LpcVertex, followed by the integer indices of the associated main curve and branch

Vertex:Position

The co-ordinate components of the LpcVertex position

NumberOfClusters

The integer number of LpcClusters found for the dataset

Cluster:Index

The integer index number of the LpcCluster

Cluster:MainCurveIndex

The integer index number of the main LpcCurve associated with the LpcCluster, followed by the LpcBranch index as well as the indices of the LpcPoints used to find the cluster

Cluster:Centroid

The co-ordinate components of the centroid position of the LpcCluster

Cluster:PrincipalAxes

The matrix (row = axis, column = co-ordinate components) of the principle component axes of the LpcCluster

Cluster:ConvexHull

The lengths along each (rotated) co-ordinate axis of the convex hull (LpcConvexHull) encompassing the boundary of hits in the LpcCluster, after the hit positions have been rotated with respect to the main principal axis of the cluster

Cluster:IsShower

Boolean to indicate if the LpcCluster has been identified as a shower

Cluster:HitIndices

The number of LpcHits in the LpcCluster, together with a list of the index of each LpcHit used in the cluster (10 hits per line)
